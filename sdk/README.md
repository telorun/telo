# Telo SDKs

Language‑specific SDKs for building Telo modules. Each SDK exposes the same conceptual contract (module lifecycle, contexts, and resource execution) while remaining idiomatic to its language ecosystem.

New to extending Telo? Start with **[Authoring a Module](../docs/extend/authoring-a-module.md)** — how a `Telo.Definition` and a controller combine to add a new resource kind — then pick your language SDK below.

- [**Rust**](rust/README.md)
